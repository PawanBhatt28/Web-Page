    TAGS

  • html   : <html> tag is the container for all other HTML elements except <!DOCTYPE html>.
  • meta   : <meta> tag defines metadata about an HTML document.
  • link   : <link> tag connects current document with external resource.
  • title  : <title> tag defines the title of the document.
  • navbar : <nav> tag defines a set of navigation links.
  • a      : <a> tag defines a hyperlink,used to link from one page to another.
  • div    : <div> tag defines a division or a section
  • body   : <body> tag defines the document's body
